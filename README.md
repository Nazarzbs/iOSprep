# iOS Interview Prep 🍏

Інтерактивні матеріали для підготовки до співбесід iOS-розробника.

**Сайт:** https://nazarzbs.github.io/iOSprep/

## Що всередині

- **[Теорія — 36 тем](https://nazarzbs.github.io/iOSprep/prep.html)** — Swift, SwiftUI, UIKit, мережа, сховище, тестування, архітектури. Кожна тема: як працює → яку проблему вирішує → де застосовується → приклад коду → пастки → контрольні питання.
- **[Лайв-кодінг — 22 завдання](https://nazarzbs.github.io/iOSprep/trainer.html)** — реальні задачі зі співбесід із підказками та розвʼязками: delegate pattern, reuse-баги, витоки памʼяті, потокобезпечність, і т.д.

Прогрес (чек-бокси) зберігається в localStorage браузера.

## Коментарі та нотатки

Під кожною темою — кнопка «💬», яка відкриває коментарі прямо на сайті (працює через [giscus](https://giscus.app)). Усі нотатки зберігаються в [Discussions](https://github.com/Nazarzbs/iOSprep/discussions) цього репозиторію. Щоб написати коментар, потрібен GitHub-акаунт.

## Як оновлювати сайт

Структура проєкту, формат тем і робочий цикл оновлення — у [STRUCTURE.md](STRUCTURE.md). Коротко: вміст теорії живе у `generator/prep_data_*.js`, `prep.html` генерується командою `node generator/gen_prep_html.js`, нові теми додаються в кінець секції.
