# iOS Interview Prep 🍏

Інтерактивні матеріали для підготовки до співбесід middle iOS-розробника.

**Сайт:** https://nazarzbs.github.io/ios-interview-prep/

## Що всередині

- **[Теорія — 36 тем](https://nazarzbs.github.io/ios-interview-prep/prep.html)** — Swift, SwiftUI, UIKit, мережа, сховище, тестування, архітектури. Кожна тема: як працює → яку проблему вирішує → де застосовується → приклад коду → пастки → контрольні питання.
- **[Лайв-кодінг — 22 завдання](https://nazarzbs.github.io/ios-interview-prep/trainer.html)** — реальні задачі зі співбесід із підказками та розвʼязками: delegate pattern, reuse-баги, витоки памʼяті, потокобезпечність, і т.д.

Прогрес (чек-бокси) зберігається в localStorage браузера.

## Локальний запуск

Просто відкрий `index.html` у браузері — жодних залежностей.
