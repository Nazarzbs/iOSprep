# iOS Interview Prep — правила для агента

Перед будь-якими змінами прочитай **STRUCTURE.md** — там структура проєкту, формат тем і робочий цикл.

Ключові правила:

- **`prep.html` НЕ редагувати руками** — він генерується. Вміст тем живе у `generator/prep_data_*.js` (секції: swift, swiftui, uikit, adjacent).
- Після зміни даних перегенеруй: `node generator/gen_prep_html.js` (пише `prep.html` у корінь).
- **Нові теми додавати тільки в кінець масиву `topics`** своєї секції — id тем порядкові (`swift-0`…), вставка в середину ламає якорі нотаток у Discussions, збережений прогрес і посилання з `schedule.html`.
- Після додавання/видалення теми онови лічильник у `title` секції (напр. `"SWIFT — 15 тем"`).
- `index.html`, `trainer.html`, `schedule.html` — статичні, редагуються напряму.
- Мова вмісту — українська; технічні терміни лишай англійською (retain cycle, dispatch тощо).
- Деплой: пуш у `main` → GitHub Actions → https://nazarzbs.github.io/iOSprep/ за 1–2 хв. Якщо деплой упав — НЕ Re-run, а новий коміт (можна порожній).
